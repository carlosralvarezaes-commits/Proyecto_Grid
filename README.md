# The Eyes Never Lie

## Autor

Carlos Alvarez

---

## Descripción de la práctica

Este proyecto consiste en el desarrollo de una página web estática centrada en la idea de que los ojos reflejan la verdadera naturaleza de las personas.

La página combina contenido visual y textual con el objetivo de transmitir un mensaje emocional. Se incluyen imágenes, texto reflexivo y contenido multimedia mediante la integración de música.

Además, se ha trabajado el diseño responsive para garantizar una correcta visualización en distintos dispositivos, especialmente en móviles.

---

## Tecnologías utilizadas

* HTML5
* CSS3
* Flexbox
* CSS Grid
* Media Queries (Responsive Design)

---

## Contenido adicional

* Integración de música mediante iframe (Spotify)
